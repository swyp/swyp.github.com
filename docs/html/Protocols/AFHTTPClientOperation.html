Overview

HTTPRequestOperationWithRequest:success:failure:

Constructs and initializes an operation with success and failure callbacks.

+ (id)HTTPRequestOperationWithRequest:(NSURLRequest *)urlRequest success:(void ( ^ ) ( id object ))success failure:(void ( ^ ) ( NSHTTPURLResponse *response , NSError *error ))failure

Parameters

urlRequest

The request used by the operation connection.

success

A block object to be executed when the operation finishes successfully. The block has no return value and takes a single argument, the response object from the request.

failure

A block object to be executed when the operation finishes unsuccessfully. The block has no return value and takes two arguments: the response received from the server, and the error describing the network or parsing error that occurred.

Discussion

Constructs and initializes an operation with success and failure callbacks.
